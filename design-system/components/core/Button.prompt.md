Pill-shaped primary action — violet gradient that lifts and glows on hover; use for the main CTA on any surface.

```jsx
<Button onClick={watch}>▶ Watch one day decide itself</Button>
<Button variant="ghost" as="a" href="#router">The four disciplines ↓</Button>
<Button variant="soft" size="sm">Replay</Button>
```

Variants: `primary` (default, violet gradient), `ghost` (transparent + hairline), `soft` (tinted violet). Sizes `sm | md | lg`. Pass `as="a"` with `href` for link buttons. Arrows (`▶ → ↓`) are added inline in the label, not as icons.
