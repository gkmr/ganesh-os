A transient notice, tone-tinted, auto-dismissing. Use for applied decisions and eval results.

```jsx
<Toast tone="success" title="Decision applied" onDismiss={pop}>done M51, stamped 7:43. Re-mirrored.</Toast>
<Toast tone="warn" title="Auto-parked 63 items">Overdue held at zero. Every move is in the change log.</Toast>
```

Tones: info (violet), success (green), warn (amber), error (red). Position with a fixed wrapper (e.g. bottom-right stack).
