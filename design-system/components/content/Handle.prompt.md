The repliable item handle from the format contract. The namespace prefix (W/M/K/V/E/I/P/PR/J) auto-tints the pill and ties the item to the daily manifest.

```jsx
<Handle id="M51" /> ship the diligence memo before noon.
<Handle id="V3" decision="push Thursday" />
<Handle id="PR2" decision="drop (confirm)" />
```

Namespaces: W/M/K chats (cyan), V voice (orange), E email (teal), I intake (amber), P day-plan (violet), PR prune (red), J job-role (amber). Decision verbs: keep · done · push <when> · drop · list <name> · p1/p2/p3 · new:.
